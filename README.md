# PCSHOP

#### 介绍
微信小程序商城，多门店+拼团+秒杀+砍价+优惠券

#### 软件架构
后端:thinkphp3.2.3
前端:未使用uniapp等第三方开发框架


#### 安装教程
小程序端：clone或者下载代码，用微信开发者工具进行编辑
         请修改相应的appid
后端:QQ群:573290500,群文件下载，后端免费
